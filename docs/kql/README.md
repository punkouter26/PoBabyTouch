# KQL Query Library for PoBabyTouchGc

This folder contains Kusto Query Language (KQL) queries for monitoring the application in Azure Application Insights.

## Usage

1. Navigate to Azure Portal > Application Insights > Logs
2. Copy and paste queries from the .kql files below
3. Adjust time ranges and parameters as needed

## Available Queries

- `health-monitoring.kql` - System health and availability monitoring
- `performance-metrics.kql` - Application performance and response times
- `business-metrics.kql` - Game-specific metrics and user behavior
